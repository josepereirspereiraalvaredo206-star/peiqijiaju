# 佩奇家具 (Peiqi Furniture)

## 项目简介
用佩奇家具配齐家具。这是一个专为家具厂商打造的 AI 室内可视化应用。该应用允许家具厂商上传其产品图册，并将这些家具无缝地融合到客户提供的室内实景照片中。通过使用先进的生成式 AI 技术，可以帮助客户直观地看到家具在自己家中的真实效果。

## 核心功能
1. **API 密钥验证机制**：
   - 使用了高级图像生成模型 `gemini-3-pro-image-preview`。
   - 首次进入应用时，会拦截并要求用户选择一个已启用计费的 Google Cloud 项目的 API 密钥，确保模型调用的合法性和可用性。
2. **厂商专属工作台**：
   - 简洁的登录界面，输入公司名称即可进入。
   - 包含“家具图册”和“室内编辑器”两个主要模块。
3. **家具图册管理 (Catalog)**：
   - 支持拖拽或点击上传多张家具图片。
   - 自动将图片转换为 Base64 格式在本地进行管理，无需配置复杂的后端存储。
   - 支持图册的预览和删除。
4. **AI 室内编辑器 (Room Editor)**：
   - **步骤 1**：上传客户的室内实景图。
   - **步骤 2**：从已上传的家具图册中点选一件家具。
   - **步骤 3**：点击“在房间中可视化”按钮，调用 Gemini API。
   - **AI 处理**：应用会将室内图、家具图以及精心设计的系统提示词（Prompt）发送给 `gemini-3-pro-image-preview` 模型。模型会识别房间结构，将选中的家具合理地放置在房间中，并处理好比例、透视、光照和阴影，生成一张逼真的合成照片。
   - **结果展示与下载**：生成完成后，可以直接在界面上预览，并提供一键下载功能。

## 技术栈
- **框架**: Next.js 15 (App Router), React 19
- **样式**: Tailwind CSS v4
- **图标**: Lucide React
- **动画**: Motion (Framer Motion)
- **AI 模型**: `@google/genai` SDK, `gemini-3-pro-image-preview` 模型

## 本地化 (Localization)
- 整个应用界面已完全汉化，面向中国市场和用户。
- 包含所有的提示信息、按钮文本、错误处理信息等。
- 传递给 AI 模型的 Prompt 也已使用中文编写，以确保模型能更准确地理解意图。
